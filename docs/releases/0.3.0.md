# swift-lm 0.3.0

`0.3.0` is the first release built around the renamed public inference surface:

- `InferenceSession`
- `PreparedPrompt`
- `PromptSnapshot`
- `GenerationParameters`
- `GenerationEvent`
- `ReasoningOptions`

## What 0.3.0 Includes

- direct Metal decode and prefill execution on Apple Silicon
- HuggingFace snapshot loading from `config.json`, tokenizer files, and `.safetensors`
- regenerable `model.staf` execution cache
- prompt preparation and reuse through `InferenceSession`
- explicit reasoning visibility control through `ReasoningOptions`
- Qwen3.5-VL and Gemma 4 prompt preparation / execution coverage in the current runtime
- Metal residency ownership contract tests for prompt snapshots, hidden override workspaces, and stable residency grouping

## Supported Scope

- Apple Silicon with a usable Metal device
- Platforms currently declared in `Package.swift`: macOS 26+, iOS 26+, visionOS 26+
- Text and chat generation through `ModelInput`
- Prompt snapshot reuse through `PromptSnapshot`
- Current multimodal execution support for the documented Qwen and Gemma paths

## Not Included in 0.3.0

- Tool calling or structured function-calling APIs
- Non-Metal execution backends
- Training or fine-tuning workflows
- Broad cross-platform validation outside the package's declared Apple platform targets

## Release Validation

The `0.3.0` release candidate was validated with:

- `xcodebuild build-for-testing -scheme swift-lm-Package -destination 'platform=macOS'`
- `xcodebuild test-without-building -scheme swift-lm-Package -destination 'platform=macOS' -only-testing:SwiftLMTests/GenerationThinkingOptionsTests`
- `xcodebuild test-without-building -scheme swift-lm-Package -destination 'platform=macOS' -only-testing:SwiftLMTests/ChatTemplateRenderingTests`
- `xcodebuild test-without-building -scheme swift-lm-Package -destination 'platform=macOS' -only-testing:SwiftLMTests/ReleaseSmokeOutputTests`
- `xcodebuild test-without-building -scheme swift-lm-Package -destination 'platform=macOS' -only-testing:MetalCompilerTests/MetalResidencyOwnershipTests`
